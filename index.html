(Learning) Representations for Planning and Control

RSS 2021 Workshop Proposal

Overview

Obtaining (dynamics) models, both from data and first principles, has been studied in various different fields for decades. Deep learning has enabled to learn such models in high-dimensional observation spaces with high predictive power. However, having an accurate dynamics model does not necessarily imply that it enables efficient reasoning for planning or control. This problem is especially dominant for long-horizon tasks with combinatorial complexity, sparse reward and challenging motion planning/control aspects. Task and Motion Planning (TAMP) approaches have developed powerful representations that are capable of solving such long-horizon tasks efficiently. In TAMP, however, these representations are usually specified by expert knowledge and it is unclear how to map sensor observations to these representations, e.g. in terms of object models, or how to learn such representations in the first place.

The goal of this workshop is to bring together experts from the fields of

  • Task and Motion Planning
  • Model Learning / System Identification
  • Computer Vision / Perception
  • Control and Feedback Motion Planning
  • (Model-based) Reinforcement Learning
to discuss how representations can be obtained that enable efficient reasoning for planning and control while being closely connected to observation spaces.

Call for Contributions

We solicit 2-3 page extended abstracts using the standard RSS template. References do not count to the page limit.

Topics of interest (but not limited to) are

  • Task and Motion Planning
  • Learning in/for task and motion planning
  • Model learning / system identification
  • Abstractions and hierarchies for Planning/Control
  • Learning abstractions for RL
  • Model-based RL
  • Vision-based models for planning and control
  • Perception for planning
  • Robot manipulation

Important dates:

Submission deadline: June 18, 2021. Anywhere on Earth
Acceptance notification: July 2, 2021

Submissions should be sent directly via email as a PDF-file. Additional video attachments are welcome.

We seek original research, late breaking results that still need discussion or work that discusses the workshop topic (with empirical data or theoretical foundation). An overlap with submitted/accepted papers is acceptable, if they have not been presented before.

Contributions will be presented live as spotlight presentations, followed by a virtual poster session. Furthermore, accepted extended abstracts (and optional video attachments) will be published on this website.
